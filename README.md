AI Agent in Python

An AI agent project while following the Boot dev AI Agent course. The purpose of this project was to gain a better understanding of how AI agents work in the background, including using Python as middleware for connecting an API call and preparing prompts to automate tasks.

Focus of this project is to build a lightweight agent that can take in user input, parse it intelligently and give useful responses whilst making sure the codebase isn't complex or hard to deal with.

Accepts user prompts
